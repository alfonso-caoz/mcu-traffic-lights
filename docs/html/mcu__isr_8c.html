ISR ( INT0_vect  )

Interrupt Service Routine for INT0: Button B1/B2.

This ISR routine detects when the Button B1/B2 was pressed, and it will set the global variable pressed_button as true just if the Current State is IDLE_VEHICLE_GREEN_PEDESTRIAN_RED.

ISR ( TIMER0_COMPA_vect  )

Interrupt Service Routine for TIMER0.

This ISR routine implements a tick counter to use it as a timer for every FSM State.

Also, this ISR routine checks if the following three conditions were satisfied to move out of the Idle State 0 (IDLE_VEHICLE_GREEN_PEDESTRIAN_RED):

  1. Current State: IDLE_VEHICLE_GREEN_PEDESTRIAN_RED.
  2. Button was pressed during this Current State.
  3. This Current State was active at least 10 seconds before commuting to another state.
